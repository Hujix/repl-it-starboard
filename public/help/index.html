Setup

  the Repl.it Starboard

Before the Starboard can start working, you need to set up the bot for your server. For this, in your server, you will need to use the setup command. This will enable the starboard in your server. If you have already set up the starboard, this will reset the settings.
The only star currently supported is [‚≠ê] - :star: on discord. (The one in the website may vary depending on where it's viewed)

The Setup Command

FORMAT: *setup <channel>

EXAMPLE: *setup #starboard

You need to be an admin in the server to use this command.

Settings

  for Repl.it Starboard

Once you've setup your server and the starboard facility has been enabled, you can use these commands to modify how starboard bot works!

The min Command

This command can be used to modify the minimum number of stars required for a message to be starred.

FORMAT: *min <number> OR *minimum <number>

EXAMPLE: *min 6 OR *minimum 6
This is set to 4 by default

The channel Command

This command can be used to modify the channel to be used as starboard.

FORMAT: *channel <channel>

EXAMPLE: *channel #announcements
This is set while using setup initially.

You need to be an admin in the server to use these command.

Information

  regarding Repl.it Starboard

There are a few commands that can be used to display certain info regarding the bot.

The serverinfo Command

This command can be used to display the current settings for the bot.

FORMAT: *serverinfo OR *guildinfo

The invite Command

This command can be used to get an invite link for the bot.

FORMAT: *invite

The leaderboard Command

This command can be used to view the server-specific leaderboard.

FORMAT: *leaderboard [page]

EXAMPLE: *leaderboard OR *leaderboard 2
The first example will display the first page while the second example will display the second page.

Anyone can use these command.
